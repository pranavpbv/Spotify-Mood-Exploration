# Spotify-Mood-Exploration



Steps to run the project:

1. Have XAMPP controller installed.
2. Download all the files from repository and the dataset from the Kaggle link.
3. Run the Python Notebook.
4. Run the Apache and MySQL servers in the XAMPP Control Panel.
5. Note the MySQL port number and change it in the project files.
6. Create a database named "Spotify_Mood_Dataset" and have two tables named search_history and user as follows:
